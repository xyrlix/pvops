# 汇流箱与电气安全巡检手册

## 一、汇流箱日常检查
1. 箱体是否密封良好，有无进水痕迹
2. 内部接线端子是否发热变色
3. 保险丝/熔断器状态
4. 防雷浪涌保护器指示窗颜色（绿色=正常，红色=失效）

## 二、直流开关操作规范
1. 断开顺序：先断负载侧，再断电源侧
2. 闭合顺序：先合电源侧，再合负载侧
3. 严禁带负荷操作直流开关
4. 操作后确认指示面板状态

## 三、电气安全注意事项
1. 光伏组串在白天始终带电，断开开关后仍需等待 5 分钟放电
2. 使用前确认万用表档位正确（直流电压档）
3. 绝缘手套、绝缘鞋等 PPE 必须穿戴
4. 两人作业，一人操作一人监护

## 四、常见汇流箱故障
- 熔断器熔断：检查对应组串电流是否异常
- 防雷器失效：更换同规格 SPD 模块
- 通讯中断：检查 RS485 接线
- 箱体温度过高：检查散热风扇、清理通风口
