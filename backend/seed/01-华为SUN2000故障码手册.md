# 华为 SUN2000 逆变器常见故障码手册

## 故障码 1：直流侧电压过高
**故障描述**：直流输入电压超过逆变器最大允许电压（1100V）
**可能原因**：
1. 组串串联组件数量过多
2. 环境温度过低导致组件开路电压升高
3. 线缆压降过小
**处理步骤**：
1. 检查组串配置，确认每串组件数量不超过逆变器规格
2. 测量开路电压，确认在逆变器允许范围内
3. 检查线缆规格是否满足载流量要求

## 故障码 2：直流侧绝缘阻抗过低
**故障描述**：PV 组串对地绝缘阻抗低于阈值
**可能原因**：
1. 组件老化或破损导致漏电
2. 连接器进水或受潮
3. 直流线缆绝缘层破损
**处理步骤**：
1. 使用绝缘电阻表测量各串对地绝缘电阻
2. 检查 MC4 连接器是否密封良好
3. 排查线缆是否有破损点
4. 如组件故障，更换同规格组件

## 故障码 3：电网电压异常
**故障描述**：并网点电压超过逆变器允许范围
**可能原因**：
1. 电网电压波动
2. 并网点距离变压器过远
3. 线径过细导致压降
**处理步骤**：
1. 记录电压波动曲线，确认是否为电网问题
2. 检查并网点接线是否牢固
3. 调整并网点或联系电网公司处理

## 故障码 4：电网频率异常
**故障描述**：电网频率超出逆变器允许范围（47.5-51.5Hz）
**处理步骤**：
1. 确认是否为电网波动
2. 等待电网恢复正常
3. 如频繁出现，联系电网公司

## 故障码 5：温度过高
**故障描述**：逆变器内部温度超过保护阈值
**可能原因**：
1. 散热风扇故障
2. 安装环境通风不良
3. 环境温度过高
4. 散热器积灰
**处理步骤**：
1. 检查散热风扇是否转动
2. 清理散热器翅片上的灰尘
3. 确认安装位置通风良好
4. 必要时安装遮阳棚

## 故障码 6：漏电流过高
**故障描述**：逆变器检测到漏电流超过安全阈值
**处理步骤**：
1. 检查组件对地绝缘
2. 检查线缆连接
3. 排查是否有设备进水

## 故障码 12：通讯中断
**故障描述**：逆变器与数据采集器通讯超时
**可能原因**：
1. RS485 接线松动
2. 通讯地址冲突
3. 采集器故障
**处理步骤**：
1. 检查通讯线缆连接
2. 确认逆变器 Modbus 地址设置
3. 重启采集器
