# 光伏组件巡检标准作业流程（SOP）

## 一、巡检周期
- 日常巡检：每日一次（远程监控）
- 例行巡检：每周一次（现场）
- 全面巡检：每月一次（含红外热成像）
- 大修后巡检：专项

## 二、日常巡检内容（远程）

### 2.1 发电性能检查
1. 对比当日发电量与理论发电量
2. 检查 PR（性能比）是否正常（≥75% 为正常）
3. 检查逆变器运行状态是否全部在线
4. 检查是否有新增告警

### 2.2 组串电流监测
1. 检查组串电流是否均衡
2. 离散率 > 5% 表示有异常
3. 定位电流偏低的组串
4. 记录异常组串位置

## 三、例行巡检内容（现场）

### 3.1 外观检查
1. 组件是否有裂纹、热斑
2. 边框是否变形或腐蚀
3. 背板是否有鼓包
4. 接线盒是否密封良好

### 3.2 电气连接检查
1. MC4 连接器是否松动
2. 汇流箱内接线端子是否紧固
3. 接地线是否连接可靠
4. 直流线缆是否有破损

### 3.3 环境检查
1. 组件表面清洁度（灰尘/鸟粪/落叶）
2. 遮挡情况（周围植物生长）
3. 支架是否松动或变形
4. 排水沟是否通畅

## 四、全面巡检内容（月度）

### 4.1 红外热成像检查
1. 对全部组件进行红外扫描
2. 温度异常点标记（比正常温度高 >10°C 为异常）
3. 热斑组件记录并安排更换
4. 汇流箱内部接点温度检查

### 4.2 I-V 曲线测试
1. 抽检比例 ≥ 5%
2. STC 条件下对比出厂参数
3. 功率衰减 > 20% 需更换
4. 记录 I-V 曲线异常特征

### 4.3 绝缘性能测试
1. PV 组串对地绝缘电阻 ≥ 1MΩ
2. 交流侧绝缘电阻 ≥ 1MΩ
3. 接地电阻 ≤ 4Ω
4. 浪涌保护器状态检查

## 五、常见问题处理

### 5.1 组件热斑处理
1. 确认热斑位置并标记
2. 拍摄热成像照片留档
3. 联系厂家或更换同规格组件
4. 更换后复测确认正常

### 5.2 组串电流异常排查
1. 用钳形电流表测量每串电流
2. 对比组串式逆变器的 MPPT 数据
3. 检查异常组串连接器
4. 排除遮挡或组件故障

### 5.3 通讯中断处理
1. 检查 RS485 接线
2. 确认设备地址设置
3. 测试通讯线缆通断
4. 重启设备

## 六、巡检记录要求
- 每次巡检填写电子巡检单
- 异常项拍照留档
- 月度巡检出具巡检报告
- 所有记录保存不少于 3 年
