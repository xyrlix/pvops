# 组串式逆变器运维指南

## 一、运行参数参考
| 参数 | 正常范围 | 告警阈值 |
|------|---------|---------|
| 直流电压 | 600-800V | <400V 或 >1000V |
| 直流电流 | MPPT 额定电流 70-100% | <50% 或 >120% |
| 交流电压 | 220V±10% | <198V 或 >242V |
| 频率 | 50Hz±0.5Hz | <49.5Hz 或 >50.5Hz |
| 温度 | <60°C | >75°C 降额 >85°C 停机 |
| PR 性能比 | >75% | <70% 警告 <60% 严重 |

## 二、MPPT 效率监测
1. 对比各 MPPT 路发电量差异
2. 同型号 MPPT 差异 <5% 为正常
3. 若某路 MPPT 显著偏低，检查对应组串
4. 阴影遮挡会导致 MPPT 追踪偏差 10-30%

## 三、组串离散率诊断
**离散率 <5%**：正常，无需处理
**离散率 5-10%**：观察，标记在案
**离散率 10-20%**：预警，安排巡检
**离散率 >20%**：严重，立即排查

排查流程：
1. 用钳形电流表测量每串实际电流
2. 对比逆变器显示值与实测值
3. 检查异常组串的 MC4 连接器
4. 测量组串开路电压确认组件数量
5. 排除遮挡或组件热斑

## 四、常见组串故障
1. 连接器烧毁（接触电阻过大）
2. 组件热斑（遮挡或老化）
3. 线缆破损（动物咬伤或机械损伤）
4. 接地故障（绝缘破损或进水）
